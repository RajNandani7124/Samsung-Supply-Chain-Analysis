# Samsung Supply Chain Analysis Dashboard

## Project Overview
This project focuses on analyzing Samsung’s supply chain operations using Power BI. 
The dashboard provides insights into inventory management, supplier performance, procurement efficiency, and delivery tracking to support data-driven decision-making.

## Objectives
- Analyze supply chain performance
- Monitor inventory levels
- Track supplier efficiency
- Identify procurement trends
- Improve operational decision-making

## Tools & Technologies Used
- Power BI
- SQL
- Excel
- DAX
- Power Query

## Dataset Information
The dataset contains information related to:
- Suppliers
- Procurement orders
- Inventory records
- Delivery status
- Product categories

## Key KPIs
- Total Orders
- Inventory Turnover
- Average Delivery Time
- Supplier Performance Score
- Procurement Cost

## Dashboard Features
- Interactive filters and slicers
- KPI cards
- Trend analysis charts
- Supplier performance visualization
- Inventory tracking dashboard

## Process Followed
1. Data Collection
2. Data Cleaning
3. Data Transformation using Power Query
4. Data Modeling
5. DAX Measure Creation
6. Dashboard Design & Visualization

## Insights Generated
- Identified high-performing suppliers
- Improved visibility into procurement delays
- Tracked inventory fluctuations
- Analyzed delivery efficiency trends

## Future Improvements
- Integration with real-time data
- Predictive analytics implementation
- AI-based demand forecasting
